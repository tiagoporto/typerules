> A toolkit that sets typography settings, vertical rhythm, and more.

_Work in Progress_

## Installation

```sh
npm i typerules@alpha
```

## Usage

```stylus
@import '~typerules'
```

## Options

### Show grid

```stylus
$debug-vertical-rhythm = true

@import '~typerules'
```

### Generate basic styles

```stylus
@import '~typerules'

set-rhythm()
```
